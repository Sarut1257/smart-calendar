# Smart Calendar

A web-based calendar and task manager built with HTML, CSS, and JavaScript.

## Features
- Add, edit, delete tasks
- Categories and time
- Search and filter
- Dashboard summary
- Export / Import JSON
- Next Task section

## Technologies
- HTML
- CSS
- JavaScript
- localStorage

## Run
Open `index.html` in browser

Live Demo: https://sarut1257.github.io/smart-calendar/
